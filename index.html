Welcome to this virtual memory game!

Rules

  • There are 16 cards and 8 pairs to find
  • Click a card to flip it over
  • Click another card to see if it matches the first one
  • Upon clicking the next card you will flip the last cards back, if they were not a match
  • Find all pairs to win!
  • Try to do it as fast as you can, using as few turns as possible (2 flips = 1 turn)

Press Start New Game above to give it a go!
